Руководство по установке программы распознавания лиц на Python 
1) Скачайте Интерпретатор Python версии не выше 3.6 на ваше устройство с официального сайта: https://www.python.org/downloads/
2) Установите Интерпретатор Python на ваш компьютер путем запуска .exe файла. Не забудьте нажать на Add Python X.X to PATH
3) С помощью сочетания клавиш Win + R, выполните открытие коммандной строки (в поле Открыть должен быть текст <cmd>
4) Проверьте, правильно ли установлен python: введите <python --version>. Если все хорошо, читайте дальше
5) Установите все нужные библиотеки с помощью этих команд:
	python -m pip install matplotlib
	python -m pip install mtcnn
	python -m pip install opencv-python
Установка модулей может занять время

Не рекомендуется запускать программу на слабых устройствах.
